# kivy-gif
Um repositório onde encontramos funções para trabalhar com gifs.

## Requirements
```
python >= 3.7

kivy >= 1.11.1

opencv-python >= 4.2.0.34
```

## How To Run
```
python3 main.py

or

py main.py
```
